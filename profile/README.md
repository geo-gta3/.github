# GTA III рЃЦрЃљрЃарЃЌрЃБрЃџрЃљрЃЊ ­ЪЉІ

рЃћрЃА рЃЮрЃарЃњрЃљрЃюрЃўрЃќрЃљрЃфрЃўрЃБрЃџрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃћрЃЌрЃЏрЃЮрЃЉрЃљ рЃњрЃљрЃЦрЃљрЃарЃЌрЃБрЃџрЃћрЃЉрЃБрЃџрЃў GTA3-рЃўрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃюрЃЎрЃБрЃЌрЃЋрЃюрЃўрЃџ рЃАрЃљрЃфрЃљрЃЋрЃћрЃЉрЃА, рЃЋрЃћрЃЉрЃњрЃЋрЃћрЃарЃЊрЃА рЃЊрЃљ рЃАрЃ«рЃЋрЃљ рЃЉрЃћрЃЋрЃа рЃАрЃљрЃГрЃўрЃарЃЮ рЃюрЃўрЃЋрЃЌрЃА ­Ъўј.
рЃљрЃЊрЃарЃћ рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў рЃћрЃарЃЌ рЃАрЃљрЃфрЃљрЃЋрЃерЃў рЃўрЃДрЃЮ, рЃарЃљрЃф рЃњрЃљрЃБрЃарЃЎрЃЋрЃћрЃЋрЃћрЃџрЃА рЃЊрЃљ рЃЊрЃљрЃЏрЃљрЃЉрЃюрЃћрЃЋрЃћрЃџрЃА рЃ«рЃЊрЃўрЃЊрЃљ, рЃЕрЃћрЃЏрЃЌрЃЋрЃўрЃАрЃљрЃф рЃЎрЃў, рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃА. рЃљрЃЏрЃўрЃћрЃарЃўрЃЊрЃљрЃю рЃљрЃАрЃћ рЃљрЃдрЃљрЃа рЃўрЃЦрЃюрЃћрЃЉрЃљ.

рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃбрЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ рЃЊрЃљ рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃў рЃЊрЃљрЃћрЃЏрЃљрЃбрЃћрЃЉрЃљ рЃљрЃ«рЃџрЃЮ рЃЏрЃЮрЃЏрЃљрЃЋрЃљрЃџрЃерЃў.

рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃў:
* gxt - Java-рЃА рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљ GTA III-рЃўрЃА *.gxt рЃцрЃљрЃўрЃџрЃћрЃЉрЃЌрЃљрЃю рЃАрЃљрЃЏрЃБрЃерЃљрЃЮрЃЊ;
* gxt-editor - JavaFX-рЃќрЃћ рЃЊрЃљрЃгрЃћрЃарЃўрЃџрЃў GTA III-рЃўрЃА *.gxt рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃарЃћрЃЊрЃљрЃЦрЃбрЃЮрЃарЃў;
